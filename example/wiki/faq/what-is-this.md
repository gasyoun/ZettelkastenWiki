# What is ZettelkastenWiki?

A static site generator for **Zettelkasten-style Markdown wikis**: a folder
of notes becomes a knowledge site with resolving wikilinks, search, SEO meta,
JSON-LD structured data and a sitemap.

It was extracted from the ORS-FAQ generator that runs a production FAQ site,
together with the invariant test suite that guards it. Start with
[[../guides/getting-started]].
